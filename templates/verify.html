Verify Signed Files Using RSA Public Key

This area is reserved to the verification of digital signed documents, using a public key. To verify a signature, just:
  • Select the resume algorith of your signature;
  • Select the original file to verify;
  • Select the signature file to verify;
  • Select the public key used in the verification;
  • Click on verify file;
Digital signatures are the best way to deploy a unique signature to your files, directly linking them only to the properties of your secret keys. That means that only the ones who have your public key can check if you signed the document. A slightly change on the original document or signature, will produce a total different output.
